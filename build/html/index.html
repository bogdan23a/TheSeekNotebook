loadJSON(layerName, layer)¶

Recursively builds a parsed version of json that each request returns

Elaboration

Parameters:
  • layerName (string) – the field that will contain the structure
  • layer (obj) – the json that needs to be parsed
Returns:

void
Return type:

None
Example: 
>>> import SEEK as S
>>> request = S.read()
>>> request.request('assays',100)
>>> request.loadJSON(request, request.json['data'])
>>> request.data
